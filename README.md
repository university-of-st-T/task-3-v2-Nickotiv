## Задача 3. Калькулятор стоимости заказа с гибкими скидками

Реализуйте функцию `calculate_total(prices, *discounts, **options)`, которая вычисляет итоговую стоимость заказа с учётом скидок и налога.

**Параметры:**
- `prices` (list of numbers) – список цен товаров (обязательный).
- `*discounts` – произвольное количество скидок в процентах (применяются последовательно к общей сумме).
- `**options` – именованные аргументы:
  - `tax` – налог в процентах (по умолчанию 0).
  - `round_to` – количество знаков после запятой для округления результата. Если `None`, округление не выполняется (по умолчанию 2).

**Возвращает:** `float` – итоговую стоимость.

### Пример
```python
prices = [100, 200, 300]
total = calculate_total(prices, 10, 5, tax=20, round_to=1)
print(total)  # 615.6
# Расчёт: сумма = 600; после скидок: 600 * 0.9 * 0.95 = 513; налог 20%: 513 * 1.2 = 615.6; округление до 1 знака → 615.6
